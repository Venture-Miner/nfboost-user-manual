LOGIN

📄️ SIGN UP

In the first access to the NFboost website, the user must create an account through Facebook, Google, Email or Cellphone number.

📄️ AUTHENTICATION

Before the user can browse the site, they need to authenticate the account with a six-digit code that were sent to the email or cell phone, from which were registered on the site.

📄️ FAVORITE SPORTS

After creating the account, the user will be redirected to a page to choose their favorite sports among several options available on the site, which they can change whenever themselves want. Once this is done, just click “Save” button.

📄️ SIGN IN

On the Sign In screen, the user must click the option which they created the account (Facebook, Google, E-mail or Cell Phone) and enter the password to access.

📄️ FORGOT PASSWORD

If any time the user forgets the login password on the NFboost website, just click “Forgot your password?” and a screen will be redirected to choose the best way to recover the password with email or phone number.
